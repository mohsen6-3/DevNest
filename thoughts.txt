Create a modern, clean UI for a Django project called "DevNest".

GLOBAL:
- Fixed top navbar (does not move on scroll)
- Minimal, academic style (like Piazza but modern)
- Use cards, spacing, and clean typography

----------------------------------

LANDING PAGE:
- Navbar:
  - Left: Logo + "DevNest"
  - Right:
    - If logged out: Sign In / Sign Up
    - If logged in: Profile avatar dropdown

- Hero section:
  - Title: DevNest
  - Short tagline
  - Buttons: Sign In / Sign Up

----------------------------------

PROFILE PAGE (public view):
- Top card:
  - Avatar, name, bio
  - Stats: number of nests, posts

- Sections (grid layout):
  - Nests joined
  - Posts

----------------------------------

PROFILE PAGE (owner view):
- Add "Edit Profile" button
- Add sections:
  - My Nests
  - Latest Posts
  - Latest Announcements
  - Latest Assessments
  - Uploaded Content
- Each section shows 3–5 items with links

----------------------------------

NEST DETAIL PAGE:
- Header:
  - Nest name
  - Description

- Tabs navigation:
  - Overview
  - Posts
  - Assessments
  - Content

----------------------------------

NEST OVERVIEW TAB:
- Show:
  - Student score (if student)
  - Latest announcements
  - Latest posts
  - Due assignments
  - Latest uploaded files

----------------------------------

POSTS PAGE:
- Layout:
  - Left sidebar (fixed width, scrollable)
  - Main content area

- Sidebar:
  - Group posts by:
    - Pinned
    - Today
    - Yesterday
    - Older dates
  - Each item shows post title
  - Highlight selected post

- Main content:
  - Post title
  - Author + date
  - Content

----------------------------------

Use reusable components:
- Card layout

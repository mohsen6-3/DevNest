# Project Name
DevNest (Anything with a “Nest” in it)

---

# Description
DevNest is a collaborative academic platform designed to enhance student learning through structured communities (“Nests”), combining Q&A discussions, Nest Content organized, assessment (maybe automated), and AI-powered feedback (Extra feature).  

Inspired by platforms like Piazza, EdStem, and Gradescope, DevNest aims to centralize learning interactions, reduce redundancy, and improve knowledge retention.

---

# Objective
- Enable structured academic collaboration  
- Provide fast, searchable access to verified answers  
- Reduce repeated questions through knowledge reuse  
- Encourage self-learning and peer support  
- Foster a sense of community within courses  
- Track and analyze student progress over time  

---

# Features

## Community System (Nests)
- Create and manage course-based communities  
- Role-based access:  
  - Instructor  
  - Assistant  
  - Student  
- Join requests with approval flow  

## Q&A System
- Ask questions (threaded)  
- Answer and comment  
- Instructor-approved answers (verified)  
- Voting system (upvote/downvote)  
- Tagging + filtering  
- Searchable  

## Organized Content system
- Searchable content  
- Filter content  

## Assessment System
- Create tests (MCQ / coding)  
- Submission tracking  
- Deadline management  

## Notifications System (can be customized)
- Real-time or async:  
  - New answers  
  - Announcements  
  - Grades released  
- User-controlled preferences  

## Student Progress & Recognition System
- Submission history  
- Performance analytics  
- Dynamic student title based on activity  

## Auto-Grading System (Hopeful Feature)
- Upload Python files  
- Execute in sandbox  
- Compare output vs expected  

## AI-Powered Features (Another Hopeful Feature)
- Generate:  
  - Periodic learning summaries  
  - Weekly exam based on students qestions  
  - Weakness detection  
- Suggest:  
  - Topics to review  
  - Related questions

# User Stories

## 1- Site-Staff
- As a site staff, I can approve or reject Nest creation requests  
- As a site staff, I want to manage all Nests’ technical issues  
- As a site staff, I want to check contact us messages and reply to them  
- As a site staff, I want to check reports and be able to access where that report came from  
- As a site staff, I shall be able to monitor Nests and Users when I need to  
- As a site staff, I want to track overall system performance so that I can improve the platform  

---

## 2- Nest-Staff

### A - Instructor
- As an instructor, I want to create and manage course content so that students can learn effectively  
- As an instructor, I can approve or reject Students’ requests to join Nest  
- As an instructor, I want to monitor students’ progress so that I can evaluate their performance  
- As an instructor, I want to create assessments so that I can test students’ understanding  
- As an instructor, I want to interact with students through Q&A so that I can support their learning  
- As an instructor, I want to be able to send an announcements and students be notified  

### B - Assistant
- As an assistant, I want to help answer students’ questions so that I can support the instructor  
- As an assistant, I want to manage discussions so that the learning environment stays organized  
- As an assistant, I want to assist in content management so that the platform stays updated  

---

## 3- Nest_Member
- As a Member, I want to join a community so that I can access learning resources  
- As a Member, I want to ask and answer questions so that I can improve my understanding  
- As a Member, I want to view learning content clearly so that I can study effectively and search and filter through them  
- As a Member, I want to receive notifications so that I stay updated with new changes (announcements and Q&A updates. Can be turned off)  
- As a Member, I want to track my progress so that I can improve my performance  
- As a Member, I want to receive AI-based reviews so that I can evaluate my learning
